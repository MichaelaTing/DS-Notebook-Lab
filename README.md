# PyData Notebook Lab

This project is a hands-on tutorial collection for learning three core Python data science libraries through Jupyter notebooks.

## What You Will Learn

- **NumPy**: Array creation, vectorized operations, and numerical computation basics.
- **Pandas**: Data loading, cleaning, transformation, and analysis workflows.
- **Matplotlib**: Data visualization fundamentals, chart styling, and plotting practices.

## Project Structure

- `Numpy.ipynb`
- `Pandas.ipynb`
- `Matplotlib.ipynb`

## Who This Is For

- Beginners starting with Python data science.
- Students preparing for data analysis projects.
- Anyone who wants quick, practical notebook-based examples.

## How To Use

1. Open the project in VS Code.
2. Launch each notebook with a Python kernel.
3. Run cells from top to bottom and modify examples to experiment.

## Goal

Build a solid foundation in Python data science by practicing with clear, focused notebook tutorials.
